# AULA 06 - 02/03/2026
	Slide Aula 03 - Usabilidade, UX, acessabilidade e comunicabilidade
	Conceitos básicos na IHC
		Interação, interface e affordance(ALGO DEVE SER USADO DE JEITO INTUITIVO, SEM INSTRUÇÕES).
		Qualidade em IHC
			É subjetiva. Usar um sistema interativo significa interagir com sua interface para alcançar objetivos em determinado contexto de uso. A interação e a interface devem ser adequadas para que os usuários possam aproveitar ao máximo o apoio computacional oferecido pelo sistema. Que características a interação devem ter para serem consideradas adequadas?
		Caracteristicas:
			Usabilidade: Conjunto de fatores que qualificam quão bem uma pessoa pode interagir com um sistema interativo (Nielsen, 1993).
			Fatores da usabilidade, 
				Facilidade de aprendizado: Utilizar a pergunta do slide. Para saber se temos uma facilidade no aprendizado, se é algo que demanda de um tempo do usuário aprender, estamos pecando neste ponto
				Facilidade de recordação: Esforço cognitivo do usuário necessário para lembrar como interagir com a interface do sistema interativo, conforme aprendido anteriormente. Ações que fazemos, se é algo que tem que ser rotineiro, não invente história.
				Eficiência: Tempo necessário para conclusão de uma atividade como apoio computacional. Neste caso, devemos levar em conta uqe o usuário é preguiçoso, com isso em mente, precisamos reduzir os passos ou atividades necessários para executar uma tarefa
				Segurança no uso: Grau de proteção de um sistema contra condições desfavoráveis ou até mesmo perigosas para os usuários. Temos como por exemplo. Reduzir o risco de botões serem incorretamente acionados(não colocar o botão de sair próximo do de salvar). 
				Satisfação do usuário: Seria o grau de atendimento das necessidades do usuário quando um produto ou sistema é usado em um contexto específico. Temos como exemplo a questão de uma feedback positivo ao usuário.
				Não teremos softwares utópicos, onde atenderemos em 100% todos os fatores da usabilidade. Não teremos uma utopia, devemos focar nas funcionalidades.
